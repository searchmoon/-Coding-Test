# 문제2. 배열의 내장함수
### <pass>부분에 배열 내장함수를 이용하여 코드를 입력하고 다음과 같이 출력되게 하세요.

```jsx
**데이터**
var arr = [200, 100, 300];
//pass
console.log(arr);

**출력**
[200, 100, 10000, 300]
```
  
### 풀이
```
arr.splice(2, 0, 10000); // (2번 인덱스에, 0개를 삭제하고, 10000을 추가한다.)
  
console.log(arr);
[200, 100, 10000, 300] //를 출력한다.
```
